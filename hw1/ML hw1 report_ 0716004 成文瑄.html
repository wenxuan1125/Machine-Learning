  7. Comparison & Conclusion

    • Comparing the two validation methods, holdout validation sometimes gets better performances. However, in average, K-fold validation has better performances.
    • For mushroom dataset:
      • The performance without applied Laplace smoothing is better than that applied Laplace smoothing. After applying Laplace smoothing, there are a lot of False Negative ( those datas whose true label are ‘poisonous’, but are predicted as ‘eidible’), which lowers the accuracy.
      • It shows that Laplace smoothing gives no guarantee to have a better performance. It depends on datas.
    • For iris dataset:
      • The model has better accuracy on class ‘Iris-setosa’ than on other classes, which can be found in my performance figure in ‘6. Results’. Both precision and recall are 1 no matter which validation method is used.
      • Although there are only 150 samples in this dataset ( compared to the mushroom dataset, in which there are 8124 samples, 150 is a very small number), it still has a quite good total accuracy (>0.9). That the data (feature) fits the Gaussian distribution well is the reason. ( it can be observed in pdf figure in ‘8. Question’.)
