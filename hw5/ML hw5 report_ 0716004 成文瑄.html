    • Explain the reasoning of your model choice, data augmentation, and training process.

      • My ANN model is simple and only have two layers, which are connected sequentially.

        • The first one is an input relu layer. It change the data dimension from 3000 to 32 (for one sample).

        • Because the predict labels are categorical labels, I use ‘categorical_crossentropy’ as loss function.
